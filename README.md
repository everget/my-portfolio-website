# My Portfolio Website

A personal developer portfolio built with React 19 and Vite. Features a skills showcase, project gallery, responsive mobile-friendly layout, dark/light theme, and full i18n support across 17 languages (EN, RU, UK, DE, FR, ES, IT, PT-BR, PL, JA, ZH, AR, HE, TR, HI, HY, KA).

[Live demo](https://everget.github.io/my-portfolio-website/)
